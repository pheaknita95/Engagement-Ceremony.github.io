# Wedding Invitation – GitHub Pages

## Quick deploy
1. Create a new GitHub repository (Public recommended).
2. Upload **index.html** from this folder to the repository root.
3. Go to **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: **main** / Folder: **/(root)** → **Save**
6. Wait 30–120 seconds, then open the URL GitHub shows.

## Admin Panel
- Open Admin Panel: **Long-press on the logo/left area** or **tap 5 times**
- Password is set inside the HTML.
- Settings + admin auth are stored in your browser (localStorage).

## Guest name
Add `?guest=NAME` to the URL, e.g.:
`https://username.github.io/repo/?guest=លោក%20សុខា`

## Copy Link with Guest
Use Admin Panel field 'Guest Name' then click Copy Link.
